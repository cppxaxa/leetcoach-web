function lis(arr):
    n = length(arr)
    dp[0..n-1] = 1
    parent[0..n-1] = -1
    for i = 1 to n-1:
        for j = 0 to i-1:
            if arr[j] < arr[i] and dp[j] + 1 > dp[i]:
                dp[i] = dp[j] + 1
                parent[i] = j
    return max(dp)

Explanation

Algorithm Intuition: For each element, we track the length of the longest increasing subsequence ending at that position by checking all previous smaller elements.

Key Idea: Dynamic programming builds solutions incrementally using previously computed values.

Time Complexity: O(n¬≤) where n is the length of the sequence.

Space Complexity: O(n) for storing DP and parent arrays.
